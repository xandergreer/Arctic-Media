## 🎨 Usage Guide

### For iOS Development
1. In Xcode, open your Assets.xcassets
2. Select AppIcon
3. Drag and drop each `ios-app-icon-*.png` to its corresponding size slot
4. Xcode will automatically match by size

### For Roku Development
1. Use `roku-app-icon-540x405.png` for HD channels
2. Use `roku-app-icon-290x218.png` for SD channels
3. These are square versions optimized for app icons

### For Website/Marketing
- Use SVG files whenever possible (infinitely scalable, small file size)
- `arctic-logo-horizontal.svg` - Best for website headers
- `arctic-logo-stacked.svg` - Best for splash screens, social media
- `arctic-logo-compact.svg` - Best for navigation bars

## 🎯 Color Palette

- **Gradient**: #5bc0ff → #2a6bff (light blue to deep blue)
- **Ice Cap**: #EAF5FF (light ice blue, 95% opacity)
- **Secondary Text**: #9ec8ff (medium blue)
- **Primary Text**: #ffffff (white)

## 📐 Design Notes

- The "A" shape represents a mountain/iceberg with the Arctic theme
- Ice cap triangle at the top adds a distinctive element
- Inner triangle creates depth and improves recognition at small sizes
- Gradient gives the logo a modern, premium feel
- Works best on dark backgrounds (#0a0e27 to #1a1f3a range)
